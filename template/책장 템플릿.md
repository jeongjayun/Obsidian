## 🟩 다 읽은 책
```dataview
TABLE without id
	status as "상태",
	file.link as "도서명",
	category as "카테고리",
	total_page as "전체 페이지", 
	dateformat(start_read_date, "DD") as "시작일",
	dateformat(finish_read_date, "DD") as "완료일",
	(finish_read_date - start_read_date + dur(1days)).days as "기간",
	my_rate as "내 평점",
	book_note as "서평"
FROM #📔메모/🌰씨앗/📚독서  %% 변경해주세요! %% 
SORT finish_read_date DESC
WHERE status = "🟩 완료"
```
