## 반복문 시작
특정 코드를 반복해서 실행할 때 사용한다. <br>
자바의 반복문에는 3가지 종류가 있다.

# while 1
```java
package loop;  
  
public class While1_1 {  
    public static void main(String[] args) {  
        int count = 0;  
  
        count = count + 1;  
        System.out.println("현재 숫자는 : " + count);  
        count = count + 1;  
        System.out.println("현재 숫자는 : " + count);  
        count = count + 1;  
        System.out.println("현재 숫자는 : " + count);  
        count = count + 1;  
        System.out.println("현재 숫자는 : " + count);  
    }  
}
```
